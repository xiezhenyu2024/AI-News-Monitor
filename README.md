## 三、推送

通过 **PushPlus** 发送到你的微信服务通知。

没收到消息：
1. 微信搜索「PushPlus 推送加」并关注
2. 在里面查看历史推送记录
3. 开启「接收文章推送」

---

## 四、查看运行状态

https://github.com/xiezhenyu2024/AI-News-Monitor/actions

- 推送是否成功
